Buat API Key baru
Di halaman credentials Roblox:
1
Klik Create API Key
2
Beri nama bebas, contoh: "Audio Uploader"
3
Di Access Permissions → pilih Assets → tambah Read & Write
4
Klik Save & Generate Key
5
Copy API Key yang muncul (hanya tampil sekali!)
